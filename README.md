Service Management Portal

A full-stack Service Management Portal that helps customers, administrators, and technicians efficiently manage product service requests, technician assignments, service charges, parts management, and ticket tracking.

🚀 Features

Customer Features

- Raise service tickets for products.
- View all submitted tickets.
- Track live ticket status updates.
- Receive automatic email notifications.
- Download service reports and invoices in PDF format.

Admin Features

- Manage all service tickets.
- Create and manage customers.
- Assign roles (Customer / Technician).
- Assign technicians to service requests.
- Monitor service progress and ticket history.
- View all customers, technicians, and tickets.

Technician Features

- View assigned tickets.
- Update ticket status.
- Add service charges.
- Add replacement parts.
- Add repair notes and remarks.
- Complete service requests.

Parts Management

- Separate parts management module.
- Add parts with quantity and price.
- Automatic parts cost calculation.
- Automatic total service cost calculation.

Notifications & Reports

- Automatic email notifications.
- Real-time ticket tracking.
- PDF report and invoice generation.

🔄 Workflow

1. Customer creates a service ticket.
2. Admin reviews the ticket.
3. Admin assigns a technician.
4. Technician performs the service.
5. Technician updates ticket status.
6. Technician adds service charges and parts.
7. System automatically calculates total cost.
8. Customer receives email notifications.
9. PDF report/invoice is generated.

Frontend Setup

cd frontend
npm install
npm run vite

Frontend URL:

http://localhost:5173

Backend Setup

cd backend
npm install
npm start

Backend URL:

http://localhost:5000
